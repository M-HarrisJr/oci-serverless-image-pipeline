# Serverless Image Processing Pipeline (OCI)

This project is a fully serverless image-processing workflow built on Oracle Cloud Infrastructure (OCI).  
It processes user-uploaded images, automatically generating thumbnails and optimized versions using OCI Functions.

## Features
- Serverless Node.js function
- Automatic processing triggered by Object Storage upload events
- Public API exposure via API Gateway
- Infrastructure built with VCN, subnets, IAM policies, and event rules

## Tech Stack
- OCI Functions (Node.js)
- API Gateway
- Object Storage
- Events + Logging
